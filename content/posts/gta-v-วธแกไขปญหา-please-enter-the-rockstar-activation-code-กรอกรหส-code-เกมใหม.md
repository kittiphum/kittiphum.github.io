Please enter the Rockstar Activation Code in order to activate Grand Theft Auto V วิธีการแก้ปัญหานั้น ง่ายๆมากคือ ปัญหาคืออินเตอร์เน็ตของคุณครับ ไม่ได้เกิดจากที่ไอดี GTA V ของคุณโดน hack หรือมีคนแอบเอาไอดีไปใช้เกิดจากที่เน็ตของคุณ เพราะว่าเวลาเปิดเกม Rockstar มันจะต้องเชื่อมกับ server ของ Rockstar เอง เมื่อมันเชื่อมไม่ได้มันก็เลยขึ้นข้อความหรือรูปแบบนี้ วิธีแก้ง่ายๆคือ

1. ไปดูว่าเน็ตดีหรือยังอาจจะต้องลองรีสตราเร้าเตอร์ดู

2. โทรไปคอลเซ็นเตอร์ครับให้เขารีเซ็ตสัญญาณเน็ต

ประสบการณ์ส่วนตัว

เคยใช้เน็ตของทรูมีปัญหาในส่วนนี้ ได้ทำการโทรไปคอลเซ็นเตอร์ แล้วบอกไปว่า เล่น GTA V ออนไลน์ไม่ได้ให้รีเซ็ตสัญญาณเน็ตให้หน่อย ปรากฏว่าเขารีเซ็ตให้ปัญหานี้จึงหายไป และเนื่องจากเห็นปัญหานี้ในกลุ่ม GTA V ใน Facebook บ่อยมากและส่วนมากจะเจอกับปัญหากับเน็ตทรูเป็นส่วนใหญ่
