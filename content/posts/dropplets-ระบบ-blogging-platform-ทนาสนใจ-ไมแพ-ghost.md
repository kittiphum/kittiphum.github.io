วันนี้มาแนวแปลกหน่อยปกติผมเองว่างๆก็ศึกษาการทำเว็บไซต์ด้วยตัวเอง เมื่อไม่มีคนสอนก็ต้องค้นด้วยตัวเอง วันนี้จะพูดถึงระบบบล็อก ระบบบล็อกตอนนี้ก็มีหลากหลายครับ ไม่ว่าจะ blogger ซึ่งตอนนี้ผมใช้อยู่ wordpress อะไรพวกนี้ก็นิยมกันหรือไม่ก็ Tumblr ก็เขียนบล็อกได้นะ ซึ่งผมเองสนใจในด้านการดีไซน์แบบเรียบๆแบบเว็บ medium.com ซึ่งเป็นแบบแหล่งที่เปิดให้กับ นักอ่านและนักเขียน เป็น social network ในรูปแบบหนึ่ง

ผมชอบดีไซน์เว็บแนวนี้มากเลยค้นไปค้นมาเจอ ghost เป็น platform CMS เขียนบล็อกเหมือนกัน คล้ายกับ wordpress แต่เพียงว่ามันต้องติดตั้งบน Node.js ซึ่งผมเองใช้ไม่เป็นหรอกครับ ซึ่งเขาบอกว่า ghost เป็นระบบเพียงแค่เขียนบล็อกเท่านั้นจริงๆมันจึงเรียบง่าย และผมว่าระบบ Node.js ยังไม่เหมาะกับบุคคลทั่วไปนะ เพราะโฮสที่ให้เช่าส่วนใหญ่จะเป็นแบบ php กันซะส่วนมาก พอนีมาพูดถึง dropplets ผมก็ค้นเจอในกูเกิ้ลอีกนั้นแหละ

สโกแกนเขาเขียนไว้ว่า Easier way to Blog ก็คงจะเขียนง่ายๆ แถมระบบยังเขียนด้วย PHP อีกต่างหากและยังไม่พอ "ไม่ต้องใช้ฐานข้อมูล" อะไรจะขนาดนั้น http://dropplets.com การเขียนบล็อกก็เพียงแค่โหลดสคริปมาแล้วอัพโหลดขึ้นโฮสติ้งเพียงแค่นั้น ในการ setup ก็เพียงแค่กำหนดรหัสผ่านในขึ้นตอนแรกก็ใช้งานได้เลยทันที การเขียนบล็อกใช้ระบบ Markdown ในการเขียนครับ (ผมยังเขียนไม่เป็น)
